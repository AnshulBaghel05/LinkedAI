# Complete Features Guide

## Free Plan Features

### 1. AI Content Generation
- Generate LinkedIn posts using Google Gemini AI
- 50 AI credits per month
- Multiple content types (tips, stories, polls, etc.)
- Tone customization (professional, casual, inspiring)

### 2. Template Library
- 12+ pre-built templates
- Categories: Tips, How-to, Statistics, Personal
- Customizable variables
- Save custom templates

### 3. Draft Management
- Save unlimited drafts
- Edit and refine content
- Convert drafts to posts
- Auto-save functionality

### 4. Content Calendar
- Visual calendar view
- Drag-and-drop scheduling
- Month/week/day views
- Color-coded post status

### 5. Post Scheduling
- Schedule posts for future dates
- Queue management
- Auto-posting via cron jobs
- Timezone support

### 6. Notifications
- Real-time activity feed
- Post published alerts
- Engagement milestones
- System notifications

### 7. Support System
- Create support tickets
- Track ticket status
- Priority levels (low/medium/high/urgent)
- Email notifications

### 8. Settings & Profile
- Update personal info
- Manage LinkedIn accounts
- Customize preferences
- Billing management

- **Overview Cards**

  - View trends over 7/14/30/90 days
  - Filter by metric (views, likes, etc.)
  - Visual bar charts

  - Sortable columns
  - Filter by date range
  - Export to CSV

- **Sync Analytics**
  - Manual sync button
  - Auto-sync every 24 hours
  - LinkedIn API integration

  - Set test duration

- **Track Results**
  - Real-time performance metrics
  - Statistical significance
  - Winner determination

- **Variants Management**
  - Edit/delete variants
  - Pause/resume tests

## How to Upgrade

1. Go to **Settings** → **Billing**
2. Click **Upgrade Plan**
3. Choose plan (Starter/Pro/Enterprise)
4. Enter payment details (Razorpay)
5. Confirm subscription
6. Features unlock immediately!
